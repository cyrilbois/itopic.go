```
pip install distance -i  https://pypi.douban.com/simple
```

# 三、示例

## 3.1 编辑距离

```
distance.levenshtein("lenvestein", "levenshtein")
```

## 3.2 汉明距离

```
distance.hamming("hamming", "hamning")
```

## 3.3 杰卡德相似系数

```
distance.jaccard("decide", "resize")
```
